### Gitflow 

_Normal workflow_

* git pull
* git checkout develop
* git checkout -b nombre_rama
* git add -A
* git commit -m "feat(commit_title): commit_description"
* git push

_Traerse cambios de develop:_

* git checkout develop
* git pull
* git checkout rama_en_la_que_esten_trabajando
* git merge develop
* git push
